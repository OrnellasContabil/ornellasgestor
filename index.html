Dúvidas frequentes

Antes de assinar, o que os contadores mais perguntam

Preciso instalar algum programa no computador do escritório?+

Não. O acesso é feito direto pelo navegador, tanto para a equipe do escritório quanto para os seus clientes no portal deles.

Meus clientes vão ter dificuldade para usar o portal?+

O portal do cliente foi pensado para ser simples: poucas abas, com nome claro do que cada uma faz (Documentos, Faturamento, Certidões, Constituição e Docs Mensais).

Consigo colocar minha própria marca no sistema?+

Sim. Cada escritório assinante tem seu próprio ambiente, separado dos demais, e a personalização com sua logo e identidade visual é configurada dentro do sistema após a assinatura.

Quantas empresas clientes posso cadastrar?+

O plano Escritório foi desenhado para a operação normal de um escritório contábil. Se você atende um volume muito grande de empresas, fale com a gente para alinhar o melhor formato.

Existe fidelidade ou contrato de permanência?+

As condições comerciais são combinadas diretamente com a nossa equipe no momento da contratação — fale com a gente pelo WhatsApp para todos os detalhes.

Como funciona o suporte se eu tiver dúvidas?+

Você recebe apoio para configurar o seu ambiente inicial e pode falar com a gente sempre que precisar de ajuda para usar qualquer módulo do sistema.
